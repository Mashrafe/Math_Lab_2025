Code 1 — Pythagorean Triples (Fortran)

প্রশ্ন: তিনটি ধনাত্মক পূর্ণসংখ্যা a, b, c যেখানে a < b < c — পিথাগোরীয় ত্রয়ী হবে যদি a2 + b2 = c2 হয়। একটি ফ্লোচার্ট/অ্যালগরিদম দেখাও যা a, b < 25 এর জন্য সকল পিথাগোরীয় ত্রয়ী নির্ণয় করবে। 

      PROGRAM MAIN   
      INTEGER A,B,C
      PRINT*, 'A B C'
      WRITE(*,*)
      DO 15 A=1,24
         DO B=A,24
            D = SQRT(REAL(A**2 + B**2))
            C = INT(D)
            IF (A**2 + B**2 .EQ. C**2) THEN
               PRINT*, A, B, C
            ENDIF
         ENDDO
15    CONTINUE
      STOP
      END PROGRAM

Code 2 — Summation with Factorials (Fortran)

প্রশ্ন: হরবৃদ্ধি ধার প্রয়োগ করে নিচের ধারার যোগফল নির্ণয় করঃ
1 + 1/2! + 1/3! + 1/4! + … + 1/13! 

      PROGRAM MAIN 
      INTEGER FACT
      SUM = 0.0
      DO 40 K = 1, 13
         FACT = 1
         DO 15 J = 1, K
            FACT = FACT * J
15       CONTINUE
         SUM = SUM + J / REAL(FACT)
40    CONTINUE
      PRINT*, 'SUM = ', SUM
      STOP
      END PROGRAM

Code 3 — Laplace Transform (Wolfram Language)

Part A — সমীকরণ সেটআপ, Laplace ও শর্ত বসানো

(* 5(a). Solve using Laplace Transform: y'' - 3 y' + 2 y = t^2, y(0)=2, y'(0)=1 *)

eq = y''[t] - 3 y'[t] + 2 y[t] == t^2;
tm = LaplaceTransform[eq, t, s];
tn = Solve[tm, LaplaceTransform[y[t], t, s]] /. {y'[0] -> 1, y[0] -> 2};

Part B — Y(s) থেকে উপসংহার

tp = tn[[1, 1, 2]];  (* Y(s) *)

Part C — বিপরীত Laplace নিয়ে y(t) পাওয়া

InverseLaplaceTransform[tp, s, t]

Code 4 — NDSolve & Plot (Wolfram Language)

(* 5(b). Solve d^2 y/dx^2 + .3 dy/dx + sin y = 0, y'(0) = 1,
   y(0) = -2, -1, 0, 1, 2,  0 <= x <= 30 and sketch the solutions. *)
Do[
  s = NDSolve[{y''[x] + .3 y'[x] + Sin[y[x]] == 0, y[0] == i, y'[0] == 1},
              y[x], {x, 0, 30}];

  f[x_] = s[[1, 1, 2]];
  g[i]  = Plot[f[x], {x, 0, 30}],
  {i, -2, 2}
];

Show[{g[-2], g[-1], g[0], g[1], g[2]}, PlotRange -> All]
